# How Much Life You've Already Used

A cinematic, interactive web experience that helps users reflect on how much of their life has already passed.

## Features
- 9 cinematic scenes with smooth transitions
- Life dot grid visualization (each dot = 1 month)
- Circular progress indicator
- Timeline breakdown (sleep / screens / work / free time)
- Web Audio API: ticking clock + heartbeat sounds
- Particle background + vignette effects
- Glitch animation on "WAIT" scene
- Fully responsive

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

## Tech Stack
- React 18
- Web Audio API
- HTML5 Canvas
- CSS Keyframe Animations

## Purpose
This is not a website — it is a life reflection experience.
It makes the user feel: *"I have already used more life than I realized."*
